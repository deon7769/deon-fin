## 1. Dados (dataset seedado — simples)
Arquivo de referência **`data/country_ratings.json`** (versionado; ratings soberanos mudam raramente →
atualização manual). Estrutura por país:
```jsonc
{
  "BR": { "name":"Brasil", "name_intl":"Brazil", "main_index":"Ibovespa",
          "ratings": { "sp":"BB", "moody":"Ba2", "fitch":"BB-" },
          "tier_label":"Médio Risco", "tier":"medium",
          "empresas":[{"name":"Vale","ticker":"VALE3","setor":"Mineração"}],
          "etfs":[{"ticker":"BOVA11","label":"ETF Brasileiro"}] },
  "US": { "name":"Estados Unidos","name_intl":"United States","main_index":"S&P 500",
          "ratings":{"sp":"AA+","moody":"Aaa","fitch":"AAA"},
          "tier_label":"AA (Alta Qualidade)","tier":"high",
          "empresas":[{"name":"Apple Inc.","ticker":"AAPL","setor":"Tecnologia"}, …],
          "etfs":[{"ticker":"SPY","label":"ETF Americano"},{"ticker":"IVV"},{"ticker":"VOO"}] }
  // … DE (AAA), IN (BBB), RU (B)… seed inicial dos países do print
}
```
**Tiers → cor (choropleth):** `top` (AAA) azul `#2563EB`; `high` (AA/A) verde `#22C55E`; `medium` (BBB/BB) âmbar
`#F59E0B`; `speculative` (B/CCC) vermelho `#EF4444`; `nodata` cinza `#3A3A3E`. (Derivar `tier` do rating S&P.)

> Sem tabela no banco (é estático). Se preferir editável depois, vira migration `country_ratings` — fora do escopo agora.

## 2. Endpoints (router `portfolio.py`)
- **`GET /api/investments/map`** → `[{ code, name, tier, color }]` (leve, para pintar o choropleth).
- **`GET /api/investments/map/{code}`** → detalhe `{ name, name_intl, main_index, ratings:{sp,moody,fitch},
  tier_label, empresas:[...], etfs:[...] }`. (Ou um único payload se o dataset for pequeno.)
- Lê o JSON em memória (cache no processo). Erros no envelope F0.1; `{code}` desconhecido → 404.

## 3. Frontend — `/investimentos/mapa` (azul)
- Nova dep: **`leaflet` + `react-leaflet`** + um **GeoJSON de fronteiras** dos países (arquivo estático em
  `web/public/world.geo.json` ou via CDN CARTO/OSM tiles como no print).
- Header "Mapa — Verifique abaixo a saúde financeira de cada País." + busca **"Buscar por país ou índice…"**.
- **Mapa** (tiles OSM/CARTO dark) com camada de **choropleth** colorindo cada país por `color` (§1). Clique no
  país (ou seleção pela busca) → carrega o painel.
- **Painel lateral** do país: nome + **selo de risco** (`tier_label`, cor do tier) e **3 abas** (azul ativa):
  - **Índices:** Principal Índice, Nome internacional, **Classificações de Risco** (S&P / Moody's / Fitch).
  - **Empresas:** lista (nome, ticker, setor).
  - **ETFs:** lista (ticker, rótulo).
- Sem `MoneyText`/Ocultar (não há valores do usuário). Estados loading/erro; mobile: painel abaixo do mapa.
- Hooks: `useMap()` (lista p/ choropleth), `useCountry(code)` (detalhe).
