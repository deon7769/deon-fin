# F1.1 Buckets Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implementar os 6 potes de Meta, classificacao automatica, propagacao por similaridade, endpoints e `BucketSelect` funcional.

**Architecture:** O backend mantem SQL em repositorios/`Database`, expondo contratos pequenos em routers FastAPI. A classificacao default vive em `src/agent/buckets.py`, perto de `Categorizer`, e a mutacao manual/propagacao vive em `transactions_repo`. O frontend mantem F0.2/F0.3: TanStack Query via hooks e controles client-side em `web/components/ui`.

**Tech Stack:** Python 3.12, FastAPI, SQLite, pytest, Next.js, React 19, TanStack Query, Vitest, Tailwind.
